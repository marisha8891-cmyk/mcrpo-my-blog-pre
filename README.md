# Бэкенд приложения-блога с использованием инструмента Spring Boot и Spring Boot Test

Бэкенд приложения для блога, построенный на Spring Boot.

# Технологический стек

- Java 17
- Spring Boot 3.2.3
- Spring Web MVC
- Spring JDBC
- H2 Database (in-memory)
- Maven
- JUnit 5, Mockito

# Сборка проекта

```bash
mvn clean package
